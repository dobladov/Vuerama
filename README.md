## Build Setup

``` bash
## Development
$ git clone https://github.com/dobladov/videorama.git
$ cd videorama
$ npm install # Or yarn install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm start

# generate static project
$ npm run generate
```
